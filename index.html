Commands

print() Outputs data to the console. E.g., print("Hello, World!") displays "Hello, World!"

if Executes code based on a condition. E.g., if x > 10: runs if x is greater than 10.

for Loops through a sequence. E.g., for i in range(5): iterates from 0 to 4.

def Defines a function. E.g., def my_function(): creates a function named my_function.
