Commands

print() - This command is used to output data to the console. For example, print("Hello, World!") will display "Hello, World!".

if - This command allows you to execute a block of code based on a condition. For example: if x > 10: will execute the following block if x is greater than 10.

for - This command is used for looping through a sequence (like a list or string). For example: for i in range(5): will iterate over the numbers 0 to 4.

def - This command is used to define a function. For example: def my_function(): creates a function named my_function.
